ЭТА ДЕРЕКТОРИЯ ПОКА ЧТО В ЭТАПЕ РАЗАРАБОТКИ

В этом файле описываю как будет язык выглядеть

  1. возможность программировать в одну строчку

for i , args echo << i - идентично - for i , args ENTER echo << i

где ENTER символ переноса строки

  1. концепция слов. “Все что не отделяется пробелом табом и прочим - одно целое слово”

math=1 - слово

math = 1 - присвение слова 1 к слову math

  1. синтаксис будет отсуцтвовать, будут только заранее зарезервированые слова

к примеру: for int / * SPACE = и прочие

  1. новый взгляд на бедные массивы

1 , 2 - 1 элемент (массив) - идентично - array { array { 1 2 } }

1 2 - 2 элемента (слова) - идентично - array { 1 2 }

.. отсуцтвие запятой между словами - знак что слова идут последовательно

.. запятая же говорить будет о том, что следующее слово входит в область преедущего слова

array { 1 2 } в массиве 2 элемента - слова

array { 1 , 2 } в массиве 1 элемент - массив (в котором 2 слова) - идентично - array { array { 1 2 } }

array { 1 2 } , 3 - идентично - array { 1 2 3 }

array { 1 2 } 3 - идентично - array { array { 1 2 } 3 }

  1. в этом языке абсолютно всё это массивы и строки.

поэтому каждый оператор кода и все что угодно в обьекте можно будет взять как элемент из массива
